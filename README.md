## Getting Started

Follow these instructions to get a local copy of the portfolio up and running on your machine:

1. **Clone the repository**:

```bash
git clone https://github.com/mwalterskirchen/mwalterskirchen.dev.git
```

2. **Navigate to the project directory**:

```bash
cd mwalterskirchen.dev
```

3. **Install dependencies**:

```bash
npm install
```

4. **Set up environment variables**:

Copy the `.env.example` file and rename it to `.env.local.` Replace the values with your own configuration, such as Notion API keys and database information.

5. **Start the development server**:

```bash
npm run dev
```

6. **Open the portfolio in your browser**:

The development server should be running at http://localhost:3000. Open your preferred browser and navigate to this URL to view the portfolio.
